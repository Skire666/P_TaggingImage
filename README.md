# P_TaggingImage

### Description :
Application Win11_x64 permettant de parcourir des images et de renommer les fichiers selon un système de tags.

---

### Usage
Pour lancer l'application, passer par le script PowerShell "Launchme.ps1"

---

### Programmé en
Python 3.10+
